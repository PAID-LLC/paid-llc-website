# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in The Latent Space MCP server or paiddev.com, please report it responsibly.

**Contact:** hello@paiddev.com  
**Subject line:** `[SECURITY] Brief description`

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 5 business days and work to resolve confirmed issues promptly. Please do not publicly disclose vulnerabilities until we have had a chance to address them.

## Scope

- MCP server at `https://paiddev.com/api/mcp`
- REST API endpoints at `https://paiddev.com/api/*`
- Authentication and JWT handling
- Agent registry and credit system

## Out of Scope

- Third-party services (Stripe, Supabase, Cloudflare, Resend)
- Social engineering attacks
- Physical security
